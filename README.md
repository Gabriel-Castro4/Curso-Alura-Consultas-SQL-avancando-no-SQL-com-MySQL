# Curso Alura - Consultas SQL: Avançando no SQL com MySQL

Este repositório documenta minha jornada de aprendizado no curso **"Consultas SQL: avançando no SQL com MySQL"** da Alura. Aqui estão registrados os scripts e exercícios práticos desenvolvidos durante as aulas, abrangendo desde consultas básicas até tópicos mais avançados de SQL.

## 📚 Conteúdo do Curso

O repositório está organizado por tópicos, cada um representado por um arquivo SQL que consolida os aprendizados daquele módulo.

### 📁 Módulos e Tópicos

#### Fundamentos e Filtros
*   `Revisão de Consultas e Filtros.sql`: Revisão geral sobre consultas e aplicação de filtros.
*   `Filtros AND, OR, NOT e IN.sql`: Uso de operadores lógicos e o operador `IN` para filtrar dados.
*   `Usando o comando LIKE.sql`: Utilização do operador `LIKE` para pesquisas por padrões em textos.
*   `DISTINCT.sql`: Aplicação do comando `DISTINCT` para remover duplicatas.
*   `ORDER BY, ASC e DESC.sql`: Ordenação de resultados com `ORDER BY` e as palavras-chave `ASC` e `DESC`.
*   `LIMIT.sql`: Limitação do número de registros retornados com `LIMIT`.

#### Funções e Agregação
*   `FUNCTIONS.sql`: Visão geral sobre funções em SQL.
*   `CONVERSAO DE DADOS.sql`: Funções para conversão de tipos de dados.
*   `DATE FUNCTION.sql`: Utilização de funções para manipulação de datas.
*   `MATH FUNCTIONS.sql`: Aplicação de funções matemáticas.
*   `CASE, WHEN, THEN, ELSE.sql`: Estrutura condicional `CASE` para lógica dentro da consulta.
*   `GROUP BY, COUNT, SUM, MAX, AVG e etc.....sql`: Uso de `GROUP BY` para agrupar resultados e funções de agregação como `COUNT`, `SUM`, `MAX`, `MIN` e `AVG`.
*   `HAVING.sql`: Utilização do `HAVING` para filtrar resultados de agrupamentos.

#### Junções (JOINs) e União
*   `INNER JOIN.sql`: Realização de junções internas (`INNER JOIN`) entre tabelas.
*   `LEFT JOIN, RIGHT JOIN.sql`: Aplicação de junções externas (`LEFT JOIN` e `RIGHT JOIN`).
*   `FULL JOIN, CROSS JOIN.sql`: Exploração de junções completas (`FULL JOIN`) e produto cartesiano (`CROSS JOIN`).
*   `UNION E FULL JOIN ADAPTADO.sql`: Uso do operador `UNION` e adaptação do `FULL JOIN` para o MySQL.

#### Tópicos Avançados
*   `SUBCONSULTAS.sql`: Criação e uso de subconsultas (queries aninhadas).
*   `VIEWS.sql`: Criação e utilização de *views* (visões) no banco de dados.
*   `RELATÓRIO DE VENDAS POR SABOR.sql`: Projeto prático para gerar um relatório de vendas agrupado por sabor.
*   `RELATÓRIO DE VENDAS VÁLIDAS.sql`: Projeto prático para gerar um relatório de vendas válidas.

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** SQL (Structured Query Language)
*   **SGBD:** MySQL

## ▶️ Como Utilizar

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/Gabriel-Castro4/Curso-Alura-Consultas-SQL-avancando-no-SQL-com-MySQL.git
    ```
2.  Navegue até a pasta do projeto.
3.  Abra e execute os arquivos `.sql` em um cliente MySQL (como MySQL Workbench ou linha de comando) para ver os exemplos e exercícios.
